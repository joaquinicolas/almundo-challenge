# Front and API
- Clone the repository
```shell
    git clone https://github.com/joaquinicolas/almundo-challenge.git
    cd almundo-challenge
```

# HOTELS API
- Install dependencies
```
cd api
yarn install
```
- Build and run the server api
```
yarn start
```

# HOTELS FRONT

- Install dependecies

```shell
    cd /front
    yarn install
```

- Running the project
```sh
    yarn start
```
